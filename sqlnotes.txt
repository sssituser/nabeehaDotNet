In order to apply the rules to the fields we use
constraints

1.null(Default Constraint):
	This constraint allow null values into the tables
2.not null:
	It will not null values to the selected field
	It can allow duplicates
3.unique :
	This constraint will not allow duplication for 
	the selected filed and it can allow one null
	value
4.primary key
	This constraint will not duplicate values
	null values.
	It creates a unique clustered Index,
	because of the table order will be changed
	as per primary key column.

	In a table we can take only one primary key.
5.check:
	This constraint will allow the data based on
	the check condition. 
	if the condition satisfies It will allow record
	otherwise it will not
6.Default :	
	This constraint will allow a default data if the
	data is not inserted into field.
